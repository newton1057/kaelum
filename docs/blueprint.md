# **App Name**: SynapseAI Chat

## Core Features:

- Chat Interface: Clean, minimalist chat interface with user messages aligned to the right and bot responses to the left.
- Message Input: Input field at the bottom for writing messages with a send button.
- New Chat: Button or sidebar option to start a new chat session.
- AI Chatbot: AI-powered chatbot that uses reasoning as a tool to generate helpful and relevant responses based on user input.
- History: Store previous chats

## Style Guidelines:

- Background color: Dark theme with primary background color #042326.
- Text and secondary details: #D2F252.
- Font: 'Inter' sans-serif for both body and headlines, offering a modern, neutral feel.
- Clean, distraction-free layout with ample spacing between messages and subtly rounded chat bubbles.
- Smooth animations for message appearance to enhance user experience.
- Minimalist icons for send, new chat, and other controls to maintain a clean interface.
